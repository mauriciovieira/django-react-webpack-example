# django-react-webpack-example

Example of a Django + React setup using webpack.
## Requirements

- node > 8
- npm > 5
- mkvirtualenv
- python > 3.6
- postgresql > 9

## Setup 

### node

- npm i

### python

- virtual environment

```
mkvirtualenv django-react-webpack-example -p python3
pip install -r requirements.txt
```

### database

```
make db
```
